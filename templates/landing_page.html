Willkommen zur Patientendaten-Analyse

Wählen Sie eine der folgenden Optionen, um fortzufahren:

Clusteranalyse ansehen Admin-Bereich

Nutzen Sie die oben stehenden Schaltflächen, um entweder die Clusteranalyse zu starten oder den Admin-Bereich zu betreten.

Admin-Bereich

Im Admin-Bereich können Sie folgende Aktionen durchführen:

  • Datensätze bearbeiten: Aktualisieren Sie die Patientendaten für eine genauere Analyse.
  • Behandlung überarbeiten: Passen Sie empfohlene Behandlungsoptionen basierend auf neuen Daten an.
  • Namensanpassungen vornehmen: Bearbeiten Sie die Namen von Krebsvariationen oder Clusterbezeichnern, um die Übersichtlichkeit zu verbessern.

Stellen Sie sicher, dass Sie über die notwendigen Berechtigungen verfügen, um diese Änderungen vorzunehmen.
